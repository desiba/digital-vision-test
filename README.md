## Technology Used
 - NestJS
 - NestJS GraphQL
 - Prisma
 - PostgreSQL as database

## Project setup

```bash
$ npm install

$ docker compose up

$ npx prisma migrate dev --name init

$ npx prisma generate
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Run tests

```bash
# unit tests
$ npm run test
```
